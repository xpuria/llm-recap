# llm-recap

A personal workspace for recapping LLM concepts and building hands-on **GenAI** and **LLM** applications.

## About

This repository collects notes, experiments, and small applications as I review and deepen my understanding of large language models and the modern GenAI ecosystem (LangChain, Pydantic, and friends).

## Contents

| File | Description |
| --- | --- |
| `langchain-pydantic.py` | Experiments combining LangChain with Pydantic for structured outputs. |

## Getting Started

```bash
# Clone the repo
git clone <repo-url>
cd llm-recap

# (Recommended) create a virtual environment
python -m venv .venv
source .venv/bin/activate

# Install dependencies as needed, e.g.
pip install langchain pydantic
```

## Goals

- Recap core LLM concepts through working code.
- Prototype small GenAI and LLM applications.
- Keep reusable patterns (structured outputs, prompting, chains) in one place.
